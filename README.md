# 音乐歌曲推荐系统 python1279

## 项目概述

本项目是一个基于大数据和Python协同过滤算法的音乐歌曲推荐系统，使用`Django`框架进行开发。

## 技术栈

- **后端框架：** Django
- **Python版本：** 3.7/3.8
- **数据库：** MySQL 5.7 或更高版本（兼容5.X版本）
- **数据库工具：** Navicat 11（推荐使用）
- **开发工具：** PyCharm 或 vs code
- **前端框架：** Vue.js

## 功能模块

- 用户管理：注册、登录、个人信息管理
- 音乐推荐：基于协同过滤算法，为用户提供个性化推荐
- 歌曲库管理：歌曲的增删改查操作
- 数据分析：用户行为数据分析

## 系统角色

- 普通用户：可进行注册、登录、查看推荐歌曲等操作
- 管理员：管理歌曲库、用户数据、系统设置等

## 运行环境

- Python 3.7 或 3.8
- MySQL 5.7 或更高版本
- Node.js（前端Vue.js运行依赖）

## 部署步骤

1. 环境准备：安装Python、MySQL、Node.js等依赖环境
2. 数据库配置：使用MySQL创建数据库，导入数据表结构
3. 后端部署：使用PyCharm或vs code部署Django项目
4. 前端部署：使用Vue.js构建前端项目，并部署到服务器

## 目录结构

```
music_recommendation_system/
├── backend/                # 后端Django项目目录
│   ├── apps/               # Django应用目录
│   ├── manage.py           # 管理脚本
│   └── ...
└── frontend/               # 前端Vue.js项目目录
    ├── src/                # Vue.js源码目录
    ├── public/             # 公共文件目录
    └── ...
```

## 核心亮点

- **个性化推荐：** 基于Python协同过滤算法，为用户推荐符合其音乐品味的歌曲
- **响应式设计：** 前端采用Vue.js框架，实现响应式网页设计，适应各种设备
- **易于扩展：** 系统模块化设计，便于后续功能扩展和优化
